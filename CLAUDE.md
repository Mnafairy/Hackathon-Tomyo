# Project Overview

A platform for Mongolian high school students to discover opportunities — projects, competitions, and achievements. The UI is in Mongolian.

## Core Concept

- **Target audience**: High school students in Mongolia
- **Purpose**: Centralized hub for finding academic/extracurricular opportunities
- **Content types**: Projects, competitions, achievements, scholarships, programs

## Main Page

The main page lists opportunities with filtering capabilities:

- **Subject filter**: Math, Science, IT, English, Art, etc.
- **Type filter**: Competition (тэмцээн), Project (төсөл), Achievement (амжилт), Program (хөтөлбөр)
- **Tags**: Searchable tags for quick categorization (e.g., STEM, олимпиад, хакатон)
- **Requirements filter**: Age range, deadline, grade level, prerequisites

## Key Features

- Browse and search opportunities
- Filter by subject, type, tags, and requirements (age, deadline)
- Each opportunity card shows: title, type, subject, deadline, age range, brief description
- Detail page for each opportunity with full info, requirements, and application links
